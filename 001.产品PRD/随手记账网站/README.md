# 随手记账网站

基于《财务记账产品需求文档.md》V1.0.1 开发的轻量个人记账工具网页版。

## 产品定位

> 「每天三秒，钱花得明白」

一款极简、无广告、隐私优先的个人记账工具。

## 功能清单

### 已完成（V1.0.1）

- ✅ **用户认证**：用户名+密码注册/登录，用户数据隔离存储
- ✅ **快速记账**：首页直接输入金额、选择分类、确认，3 步完成记账
- ✅ **收支分类**：默认 8 大类支出 + 7 类收入分类，支持自定义新增/编辑/删除
- ✅ **账单明细**：按日期分组，支持月份切换、关键词搜索、删除账单
- ✅ **统计报表**：月度总览、分类饼图 TOP5、近 6 个月趋势图、每日收支对比
- ✅ **预算管理**：设置月度总预算，首页实时显示进度与预警
- ✅ **账户管理**：现金/微信/支付宝/银行卡/信用卡等多账户，支持新增编辑
- ✅ **数据安全**：数据默认保存在浏览器 LocalStorage，按用户隔离
- ✅ **数据备份/恢复**：导出/导入 JSON 备份文件
- ✅ **数据导出**：导出当月 CSV 文件
- ✅ **记账提醒**：设置提醒时间（Web 端通过页面标题模拟）
- ✅ **主题设置**：浅色/深色主题切换
- ✅ **移动端适配**：手机优先，同时适配桌面浏览器

## 使用方法

### 本地打开

直接用浏览器打开 `index.html` 即可使用。

```bash
open index.html
```

### 本地服务器方式（推荐）

如需更完整的体验，可以使用任意本地静态服务器：

```bash
# 使用 Python
python3 -m http.server 8080

# 使用 Node.js
npx serve .
```

然后访问 `http://localhost:8080`。

### 首次使用

1. 打开应用后，进入注册页面
2. 输入用户名（至少 2 个字符）和密码（至少 6 个字符）
3. 点击注册，成功后自动跳转到登录页面
4. 使用刚注册的账号密码登录即可开始使用

### 数据说明

- 每个用户的数据独立存储在浏览器 `localStorage` 中
- 同一浏览器可支持多账号切换（通过退出登录切换）
- 清理浏览器数据会导致所有用户数据丢失，请定期使用「数据备份」功能导出 JSON
